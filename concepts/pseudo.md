# pseudo elements

Pseudo-elements are used to style specifided parts on an elements

- ::after
- ::before
- ::first-letter
- ::first-line
- ::selection

# pseudo classes

- active
- checked
- disabled
- empty
- first-child
- focus
- hover
- last-of-type
- not(selector)
- nth:child(n)
- nth-of-type(n)
- required
- target
- visited
